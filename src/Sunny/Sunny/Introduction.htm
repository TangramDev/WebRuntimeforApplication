Sunny is a disruptive, Internet-oriented open source desktop software development framework. Its main feature is to integrate Microsoft.NET, Chromium Project and IBM Eclipse three open source projects, so that .NET desktop applications can be supported in a consistent manner. NET, Java and Web software ecosystem. Sunny defines the application infrastructure and provides a one-stop solution, so that .Net desktop software applications can directly have the following three characteristics:

  1. Each application supports a software ecosystem(for .Net resources, Java resources, Web resources)
  2. Each application has the ability to interface with mainstream open source frameworks (Chromium Project, Eclipse, .NET Core, etc.)
  3. The application integrates modern browsers with full functions (full-featured Build-in WebBrowser support)
